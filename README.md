# ShibaToDo

React、Tailwind CSSで構築したタスク管理アプリです。タスクの追加・編集・削除・並べ替えを通して、ノンストレスで直感的にタスクを管理できるアプリを作成します。

タスク完了のトグル機能では、まず画面上の表示を先に更新し、その後で API 経由でデータを更新するようにしました。更新に失敗した場合は、先に変更した UI の表示を元に戻すことで、見た目と実際のデータにずれが出ないようにしています。最初はトグル操作のたびに画面が再読み込みされたり、表示の反映が遅く感じたりして使いづらさがありましたが、原因を一つずつ確認しながら修正し、不要な再読み込みが起きないように改善しました。機能を作るだけでなく、実際に使ったときの操作感や不具合の改善まで意識して取り組んだ点がこだわりです。

開発時はフロントエンドとjson-serverを分離し、
ホットリロード時の不要な再描画を防ぐ構成にしました。
